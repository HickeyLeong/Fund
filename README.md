# shell基金小工具

1. 在fund.conf配置需要的基金和指数

``` 
[fund]
160222=国泰国证食品饮料行业指数分级
090010=大成中证红利指数
530015=建信深证基本面60ETF联接A

[stock]
sh000001=上证指数
sz399001=深证成指
sh000016=上证50
sh000300=沪深300
sh000932=中证消费
sz399922=中证红利
```

2. 运行: python fund.py
